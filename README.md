- daywkDivNm(요일구분명)
- signguCode(시군구코드)
- signguNm(시군구명)
- touDivCd(관광객구분코드); {1:현지인(a), 2:외지인(b), 3:외국인(c)}
- touDivNm(관광객구분명)
- touNum(관광객수)
